# Seismic Testnet Guide

Repository ini berisi informasi dasar tentang Seismic Testnet.

## Network Details

- Chain ID: 5124
- RPC URL: https://gcp-2.seismictest.net/rpc
- WebSocket: wss://gcp-2.seismictest.net/ws
- Explorer: https://seismic-testnet.socialscan.io/
- Faucet: https://faucet.seismictest.net/

## Cara Menggunakan

1. Tambahkan RPC ke MetaMask
2. Klaim token dari faucet
3. Gunakan untuk testing smart contract
